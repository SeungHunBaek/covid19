# Covid19 Status
코로나19 감염현황 시각화 프로젝트
コロナ19の感染現況を視覚化したプロジェクト

* React: 16.9.52
* Typescript: 4.0.2
* React-Bootstrap: 0.32.24
* webpack: 4.44.1
* eslint: 7.9.0

## Install

``` bash
# Project Download 
git clone https://github.com/SeungHunBaek/covid19.git

# Project Install
npm install

```

## Start

``` bash
# Project Start 
npm start

```
